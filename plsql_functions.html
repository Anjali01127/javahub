PL/SQL Functions

Create a Function

A function is created using the CREATE FUNCTION statement.

      CREATE OR REPLACE FUNCTION get_employee_name(emp_id NUMBER)
      RETURN VARCHAR2
      IS
        emp_name VARCHAR2(100);
      BEGIN
        SELECT first_name INTO emp_name FROM employees WHERE employee_id = emp_id;
        RETURN emp_name;
      END;

Function Calling

A function can be called from other PL/SQL blocks or functions.

      DECLARE
        emp_id NUMBER := 100;
        emp_full_name VARCHAR2(200);
      BEGIN
        emp_full_name := get_employee_name(emp_id);
        DBMS_OUTPUT.PUT_LINE('Employee Full Name: ' || emp_full_name);
      END;

Recursive Function

A recursive function calls itself either directly or indirectly.

      CREATE OR REPLACE FUNCTION factorial(n NUMBER)
      RETURN NUMBER
      IS
      BEGIN
        IF n = 0 THEN
          RETURN 1;
        ELSE
          RETURN n * factorial(n - 1);
        END IF;
      END;

Drop a Function

A function can be dropped from the database using the DROP FUNCTION statement.

      DROP FUNCTION get_employee_name;
