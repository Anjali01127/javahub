MySQL Statements

CREATE DATABASE Statement

The CREATE DATABASE statement is used to create a new database in MySQL.

      CREATE DATABASE mydatabase;

DROP DATABASE Statement

The DROP DATABASE statement is used to delete an existing database in MySQL.

      DROP DATABASE mydatabase;

CREATE TABLE Statement

The CREATE TABLE statement is used to create a new table in MySQL.

      CREATE TABLE users (
        id INT AUTO_INCREMENT PRIMARY KEY,
        username VARCHAR(50) NOT NULL,
        email VARCHAR(100) NOT NULL
      );

DROP TABLE Statement

The DROP TABLE statement is used to delete an existing table in MySQL.

      DROP TABLE users;

TRUNCATE TABLE Statement

The TRUNCATE TABLE statement is used to remove all records from a table, but the table structure remains intact.

      TRUNCATE TABLE users;

ALTER TABLE Statement

The ALTER TABLE statement is used to modify an existing table in MySQL, including adding, dropping, or modifying columns.

      ALTER TABLE users ADD COLUMN age INT;

INSERT INTO Statement

The INSERT INTO statement is used to add new records into a table in MySQL.

      INSERT INTO users (username, email) VALUES ('john_doe', 'john@example.com');

SELECT Statement

The SELECT statement is used to retrieve data from one or more tables in MySQL.

      SELECT * FROM users WHERE age > 18;

UPDATE Statement

The UPDATE statement is used to modify existing records in a table in MySQL.

      UPDATE users SET age = 30 WHERE id = 1;

DELETE Statement

The DELETE statement is used to remove existing records from a table in MySQL.

      DELETE FROM users WHERE id = 1;

GROUP BY Statement

The GROUP BY statement is used to group rows that have the same values into summary rows in MySQL.

      SELECT department, COUNT(*) FROM employees GROUP BY department;

CASE Statement

The CASE statement is used to create conditional logic in SQL queries in MySQL.

      SELECT
        CASE 
          WHEN age < 18 THEN 'Child'
          WHEN age >= 18 AND age < 65 THEN 'Adult'
          ELSE 'Senior'
        END AS age_group
      FROM users;

CREATE INDEX Statement

The CREATE INDEX statement is used to create an index on a table in MySQL, which improves query performance.

      CREATE INDEX idx_username ON users (username);

CREATE VIEW Statement

The CREATE VIEW statement is used to create a virtual table based on the result of a SELECT query in MySQL.

      CREATE VIEW active_users AS SELECT * FROM users WHERE status = 'active';

Alias

An alias is a temporary name assigned to a table or column in MySQL, which is used to make column names more readable or to rename tables.

      SELECT username AS 'User', email AS 'Email' FROM users;
